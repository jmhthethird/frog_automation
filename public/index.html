Reports

  

  

    

      

        
📄 Page Title Audit

        
Analyse page titles for length, duplication, and keyword optimisation across your crawled pages.

        
      

      

        
📝 Meta Description Audit

        
Review meta descriptions for length compliance, uniqueness, and missing entries across your site.

        
      

      

        
🔢 Response Code Report

        
Identify HTTP status code distribution including 3xx redirects, 4xx errors, and 5xx server issues.

        
      

      

        
🏗️ Heading Structure Report

        
Validate heading hierarchy (H1–H6) for proper nesting, missing headings, and SEO best practices.

        
      

      

        
🔗 Canonical Tags Report

        
Audit canonical tag implementation to detect missing, self-referencing, or conflicting canonicals.

        
      

      

        
⚡ Core Web Vitals Report

        
Monitor Largest Contentful Paint, First Input Delay, and Cumulative Layout Shift metrics.

Process Automation

  

  

    

      

        
🔑 Keyword Mapping

        
Automatically map target keywords to landing pages using crawl data and search performance metrics.

        
      

      

        
↪️ Redirect Mapping

        
Generate redirect maps from old URLs to new destinations during site migrations and restructures.

        
      

      

        
🔍 Content Gap Analysis

        
Identify content opportunities by comparing your site coverage against competitor topics and keywords.

        
      

      

        
📋 Structured Data Audit

        
Validate JSON-LD and schema markup implementation across pages for rich result eligibility.
